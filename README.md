# 私域梦海

`siyumenghai.cn` 的静态网站源码。

## 文件结构

- `index.html`：首页，展示最新日报和归档入口
- `archive.html`：归档页
- `reports/`：日报详情页
- `styles.css`：共享样式
- `script.js`：阅读进度和导航高亮
- `CNAME`：自定义域名 `siyumenghai.cn`

## 部署

把本目录发布到任意静态站点服务即可，例如 GitHub Pages、Cloudflare Pages、Vercel 静态站点或宝塔/Nginx 静态目录。

如果使用 GitHub Pages，保留 `CNAME` 文件，并把域名 DNS 指向 GitHub Pages 要求的记录。
